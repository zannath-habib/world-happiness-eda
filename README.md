# World Happiness Report Analysis (Python)
## Project Overview
This project performs exploratory data analysis on the World Happiness Report 2019 dataset to examine determinants of cross-country happiness levels.
## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
## Analysis Conducted
- Data cleaning and structuring
- Ranking countries by happiness score
- Visualization of top and bottom country indicators
- Scatter plot analysis between GDP per capita and happiness
- Examination of freedom and corruption indicators
## Key Insights
- GDP per capita shows a positive but imperfect correlation with happiness.
- Social support, governance quality, and freedom significantly influence well-being.
- Economic prosperity alone does not fully explain cross-country differences in happiness.
## Conclusion
Happiness is multidimensional and influenced by economic, social, and institutional factors. This analysis highlights the importance of combining quantitative data with economic interpretation.
